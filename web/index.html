restoration-zonal-diff

v0.1.0 demo

Streaming, Monte Carlo–propagated per-zone zonal-diff over a baseline land-cover raster and one or more restoration-scenario rasters. The full library is on GitHub; this page runs a small in-browser port of the algorithm so you can see how Monte Carlo draws translate into per-zone credible intervals.
